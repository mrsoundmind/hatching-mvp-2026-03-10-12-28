**Step 1 -- Extend the personality JSONB type in shared/schema.ts:**

At line 75-80, the `personality` column type is:
```typescript
personality: jsonb("personality").$type<{
  traits?: string[];
  communicationStyle?: string;
  expertise?: string[];
  welcomeMessage?: string;
}>().default({}),
```

Add `adaptedTraits` and `adaptationMeta` fields to the type:
```typescript
personality: jsonb("personality").$type<{
  traits?: string[];
  communicationStyle?: string;
  expertise?: string[];
  welcomeMessage?: string;
  adaptedTraits?: Record<string, Record<string, number>>;
  adaptationMeta?: Record<string, { interactionCount: number; adaptationConfidence: number; lastUpdated: string }>;
}>().default({}),
```

The `adaptedTraits` shape is `{ [userId]: { formality, verbosity, empathy, directness, enthusiasm, technicalDepth } }` -- keyed by userId because personality adapts per-user.

The `adaptationMeta` shape is `{ [userId]: { interactionCount, adaptationConfidence, lastUpdated } }` -- tracks learning progress per user.

**Step 2 -- Persist adaptedTraits after reaction handler (line ~1066):**

In server/routes.ts, find the reaction handler block (around line 1064-1072):
```typescript
if (reactionData.agentId) {
  const feedback = reactionData.reactionType === 'thumbs_up' ? 'positive' : 'negative';
  personalityEngine.adaptPersonalityFromFeedback(
    reactionData.agentId,
    userId,
    feedback,
    '',

  );
```

After the `adaptPersonalityFromFeedback` call and before the `markSkillUsed` try block, add persistence:
```typescript
// Persist adapted personality traits to database
try {
  const updatedProfile = personalityEngine.getPersonalityProfile(reactionData.agentId, userId);
  const agent = await storage.getAgent(reactionData.agentId);
  if (agent) {
    const existingPersonality = (agent.personality as any) || {};
    await storage.updateAgent(reactionData.agentId, {
      personality: {
        ...existingPersonality,
        adaptedTraits: {
          ...(existingPersonality.adaptedTraits || {}),
          [userId]: updatedProfile.adaptedTraits
        },
        adaptationMeta: {
          ...(existingPersonality.adaptationMeta || {}),
          [userId]: {
            interactionCount: updatedProfile.interactionCount,
            adaptationConfidence: updatedProfile.adaptationConfidence,
            lastUpdated: new Date().toISOString()
          }
        }
      } as any
    });
  }
} catch (persistErr) {
  console.error('Failed to persist personality adaptation:', persistErr);
}
```

Note: The `as any` cast is needed because Drizzle's Partial<Agent> expects the exact JSONB shape, and we are adding new fields. This is safe because JSONB accepts any valid JSON.
