## Smart Task Detection Pipeline (Bonus — not in original v1.1 roadmap)

The old broken task extraction (AI extraction on every message, causing false positives from greetings) has been replaced with a zero-LLM-cost intent classifier pipeline:

- **classifyTaskIntent** gates all task operations (5 intent types)
- **EXPLICIT_TASK_REQUEST** → direct creation with due date/priority/duplicate check
- **USER_DELEGATION** → assigned task creation via @mention + delegation verb
- **TASK_LIFECYCLE_COMMAND** → status/priority/assignee updates, delete, queries, progress
- **ORGANIC_CANDIDATE** → deferred for future LLM extraction (depth ≥ 4)
- **NO_TASK_INTENT** → skip entirely (greetings, reactions, short messages)
- Rate limiting: 10 creates/min/user
- Completion detection on agent responses
